Contexte du projet

Cette application web a été développée dans le cadre d’un projet pédagogique portant sur la conception d’une base de données et la création d’une application web permettant d’explorer des données de migration animale.

Le site permet notamment de visualiser les déplacements de faucons crécerelles à partir de données GPS, d’explorer certaines relations entre migration et conditions météorologiques, et de consulter les fiches associées aux individus suivis.

Données utilisées

Les données utilisées dans ce projet proviennent de jeux de données ouverts et ont été exploitées uniquement dans un cadre pédagogique et académique.

Les traitements de données ont été réalisés dans le cadre des enseignements du master, notamment pour la construction d’une base relationnelle et la production de visualisations.

Réutilisation des contenus

Le code source de l’application est diffusé dans le cadre du projet universitaire. Les contenus et visualisations présentés sur ce site peuvent être réutilisés à des fins pédagogiques ou de recherche, sous réserve de mentionner les auteurs du projet.

Responsabilité

Les informations présentées sur ce site sont fournies à titre pédagogique. Les auteurs ne peuvent être tenus responsables d’éventuelles erreurs, imprécisions ou interprétations des données.
